IDM激活插件

发表于2024-02-18|更新于2024-03-05|杂项

激活

因为最近好像IDM加强了对之前的伪装序列号的检测所以最近我的IDM频繁的弹出购买界面，虽然说吧 能用是能用 但是经常弹窗感觉还是很难受.然后去Github找了一下我之前使用的破解补丁的大佬的项目 IDM-Activation-Script 然后发现现在联网激活锁定注册表的方式好像不可以的但是依然可以使用这个插件来进行重置 来使你可以继续使用

激活方法

根据Github说明上做(以下英文为cv内容)

Method 1 - PowerShell

(Recommended)

Right-click on the Windows start menu and select PowerShell or Terminal (Not CMD).
Copy-paste the below code and press enter
irm https://massgrave.dev/ias | iex
You will see the activation options, follow the on-screen instructions.
That’s all.

Method 2 - Traditional

Download the file from GitHub 或 Bitbucket
Right-click on the downloaded zip file and extract
In the extracted folder, run the file named IAS.cmd
You will see the activation options, and follow onscreen instructions.
That’s all.
意思就是说有两种下载软件的方式

  1. 直接在powershell输入 irm https://massgrave.dev/ias | iex 这一行代码 然后选择你要进行的激活方式就可以了

  2. 下载源代码然后运行IAS.cmd

我采用的是第一种，如果你的网络不太行可以使用第二种方法建议使用moeyy大佬的加速

具体对于激活方式的讲解请前往Github

具体的激活方式你可以去GitHub看 这里我使用了2重置30天试用(现在好像第一种在线激活不可用) 此方法如果你之前下载过idm需要先把idm完全卸载卸载工具 然后再下载安装，安装完成后打开激活程序，等待程序调用IDM下载重置就可以了

然后等待出现这个界面打开idm就会发现已经激活成功了

更多花絮

欸嘿 既然说到这位大佬那就不得不说大佬的另一个工具了Microsoft Activation Scripts

这是大佬开发的一款激活windows和office的工具并且还集成了许多Windows和office软件的下载地址十分贴心 有兴趣的可以去看一看
使用方式与上方一样 就不详细说了 这个工具在PowerShell调用的命令是irm https://massgrave.dev/get | iex
